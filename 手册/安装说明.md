# NJUCode 安装说明

## 概述

`njucode` 是一个基于 Textual 的终端代码助手（Code Agent），支持代码检索分析、Patch 工程修改、Skills 插件扩展、MCP 协议集成、会话管理与上下文压缩等功能。

安装包为 Python Wheel 格式（`.whl`），位于 `手册/njucode-0.5.0-py3-none-any.whl`。

---

## 环境要求

| 项目 | 要求 |
|------|------|
| Python | >= 3.10 |
| 操作系统 | Linux / macOS / Windows（WSL） |
| 网络 | 需能访问所选模型镜像站点 |

---

## 安装方式

### 方式一：从 wheel 文件安装（推荐）

```bash
# 1. 创建并激活虚拟环境（推荐）
conda create -n njucode python=3.10
conda activate njucode

# 2. 安装
pip install 手册/njucode-0.5.0-py3-none-any.whl
```

### 方式二：源码安装

```bash
# 在项目根目录执行
pip install -e .
```

---

## 安装后配置

```bash
# 1. 设置 API Key（必选）
# 从 https://www.modelscope.cn/my/access/token 获取 token
export OPENAI_API_KEY="your-api-key-here"

# 2. （可选）自定义模型与镜像
export OPENAI_BASE_URL="https://api.atlascloud.ai/v1"
export OPENAI_MODEL="deepseek-v3"
```

也可以创建 `.env` 文件放置在运行目录下：

```text
OPENAI_API_KEY=your-api-key-here
OPENAI_BASE_URL=https://api.atlascloud.ai/v1
OPENAI_MODEL=deepseek-v3
```

## 启动

```bash
# 如果通过 pip 安装，直接运行
njucode

# 如果从源码运行
python main.py
```

## 命令行接口

启动后进入 TUI 界面，支持以下斜杠命令（在聊天输入框中输入）：

| 命令 | 功能 |
|------|------|
| `/help` | 显示帮助信息 |
| `/scan` | 扫描项目文件并建立索引 |
| `/search <kw> [--regex] [--case]` | 文本检索 |
| `/symbol <name>` | Python 符号检索 |
| `/summary <path>` | 文件摘要分析 |
| `/deps <path> [--depth 1\|2]` | 依赖图分析 |
| `/recall <desc> [--top N]` | 自然语言文件召回 |
| `/impact <target> [--depth 1\|2]` | 影响面分析 |
| `/tasks [--tag TAG] [--done]` | 项目 TODO 扫描 |
| `/metrics [--top N] [--path P]` | 静态代码指标 |
| `/doctor [--verbose]` | 项目健康自检 |

键盘快捷键：

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建会话 |
| `Ctrl+H` | 显示/隐藏聊天面板 |
| `Ctrl+C` | 中断模型流式输出 |
| `Ctrl+Q` | 退出应用 |

## 安装包内容清单

Wheel 包 `njucode-0.5.0-py3-none-any.whl` 内含以下文件：

```
njucode/
├── __init__.py                      # 包声明
├── app.py                           # 主应用类（TUI 入口，main() 函数）
├── app.tcss                         # Textual CSS 样式文件
├── models.py                        # 核心数据模型（ChatMessage/ChatSession/ModelConfig）
├── state.py                         # 全局状态管理（AppState）
│
├── services/                        # 业务服务层
│   ├── __init__.py
│   ├── openai_client.py             # OpenAI 兼容流式客户端
│   ├── code_analysis.py             # 代码分析引擎（扫描/检索/符号/摘要/依赖/召回/影响面）
│   ├── code_metrics.py              # 静态代码指标（复杂度/fan-in-out/环路/热点）
│   ├── code_extractor.py            # 代码块提取器
│   ├── context_compressor.py        # 上下文压缩器（双语 token 估算/增量压缩/质量验证）
│   ├── patch_engine.py              # Patch/回滚执行引擎（生命周期/备份/应用/回滚）
│   ├── project_testing.py           # Project Doctor 项目自检系统
│   ├── settings_store.py            # 配置持久化（.nju_code/settings.json）
│   ├── task_index.py                # 任务/TODO 扫描器
│   └── runtime_tools.py             # 运行时工具函数
│
├── skills/                          # Skills 插件系统
│   ├── __init__.py
│   ├── models.py                    # Skills 数据模型（权限/状态/种类/清单/执行日志）
│   ├── registry.py                  # 技能注册中心（内置/插件加载/命令映射/启停）
│   ├── executor.py                  # 技能执行引擎（参数校验/权限检查/结果格式化）
│   ├── permissions.py               # 权限检查器
│   ├── audit_log.py                 # 审计日志（执行轨迹持久化）
│   ├── builtin/
│   │   └── __init__.py              # 内置技能注册（13 个斜杠命令即技能）
│   └── plugins/
│       └── __init__.py              # 外部插件加载目录
│
├── mcp/                             # MCP 协议集成
│   ├── __init__.py
│   ├── models.py                    # MCP 数据模型（服务器配置/连接状态/工具信息）
│   ├── client.py                    # MCP 客户端（stdio 传输/JSON-RPC 消息收发）
│   ├── manager.py                   # MCP 管理器（多服务器连接生命周期/配置持久化）
│   ├── executor.py                  # MCP 工具执行器
│   └── tool_adapter.py              # 工具适配器（外部 MCP 工具 → 内部 ToolToggle）
│
└── ui/                              # UI 界面层
    ├── __init__.py
    └── widgets/
        ├── __init__.py              # 所有 widget 与消息事件导出
        ├── chat_panel.py            # 聊天面板（消息列表/@file注入/斜杠命令）
        ├── code_viewer_panel.py     # 代码查看面板（语法高亮/编辑保存）
        ├── config_panel.py          # 配置面板（模型/镜像/API Key）
        ├── file_tree_panel.py       # 文件树面板（浏览/新建/删除/撤销）
        ├── session_panel.py         # 会话管理面板（创建/切换/重命名/导出/导入）
        ├── tools_panel.py           # Tools 工作台（检索/分析按钮与参数输入）
        ├── skills_panel.py          # Skills 面板（技能列表/开关/执行）
        ├── mcp_panel.py             # MCP 面板（服务器连接/工具管理）
        ├── patch_panel.py           # Patch 面板（diff 预览/确认/回滚）
        └── splitter.py              # 分割条组件（三栏拖拽调宽）
```

## 依赖关系

| 包名 | 最低版本 | 用途 |
|------|----------|------|
| `textual` | 0.59.0 | TUI 框架（App/Widget/CSS/事件系统） |

| `pydantic` | 2.8.0 | 数据校验与序列化 |
| `python-dotenv` | 1.0.1 | .env 环境变量加载 |
| `openai` | 1.30.0 | OpenAI 兼容 API 客户端 |
| `mcp` | 1.27.0 | MCP 协议 Python SDK |

### 可选依赖（tree-sitter 语法高亮）

```bash
pip install njucode[syntax]
```

包含 16 种语言的 tree-sitter 解析器（Python / JavaScript / TypeScript / C / C++ / Java / Go / Rust / JSON / YAML / Markdown / Bash / HTML / CSS / SQL / TOML）。
