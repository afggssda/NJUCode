# 2025-04-23 NJUCode Skills & MCP 系统改动记录

## 一、概述

本次改动实现了两个核心功能模块：

1. **Skills 系统** - 插件化技能管理，将 CodeAnalyzer 命令封装为可扩展的技能体系
2. **MCP 集成** - Model Context Protocol 客户端，连接外部 MCP 服务器使用其工具

### 2.3 权限级别

```python
class SkillPermissionLevel(Enum):
    READ_ONLY = "read_only"          # 仅读文件
    MODIFY_LOCAL = "modify_local"    # 可修改本地文件
    EXECUTE_COMMAND = "execute_command"  # 可执行shell命令
    NETWORK_ACCESS = "network_access"    # 可访问网络
    FULL_ACCESS = "full_access"      # 完全权限
```

权限映射：
- `READ_ONLY` → `read_file` tool
- `MODIFY_LOCAL` → `write_file` tool
- `EXECUTE_COMMAND` → `terminal` tool
- `NETWORK_ACCESS` → `web_fetch` tool
- `FULL_ACCESS` → `git` tool

### 5.2 MCP 配置

```json
{
  "mcp": {
    "servers": {
      "filesystem": {
        "server_id": "filesystem",
        "name": "File System",
        "transport": "stdio",
        "command": "uvx",
        "args": ["mcp-server-filesystem", "--root", "."],
        "enabled": true,
        "auto_connect": true
      },
      "github": {
        "server_id": "github",
        "name": "GitHub",
        "transport": "stdio",
        "command": "uvx",
        "args": ["mcp-server-github"],
        "env": {
          "GITHUB_TOKEN": "${GITHUB_TOKEN}"
        },
        "enabled": true,
        "auto_connect": false
      }
    },
    "tools": {
      "mcp.filesystem.read_file": {
        "enabled": true,
        "usage_count": 0
      }
    }
  }
}
```

## 六、使用说明

### 6.1 Skills 使用

```bash
# 在聊天中输入命令
/scan              # 扫描项目
/search keyword    # 搜索文本
/symbol ClassName  # 搜索符号
/summary path.py   # 文件摘要
/deps path.py      # 依赖分析
/recall 查找测试文件  # 自然语言召回
/impact path.py    # 影响分析
/help              # 显示帮助
```

### 6.2 MCP 使用

1. 安装 MCP SDK：`pip install mcp>=1.27.0`
2. 配置服务器在 `.nju_code/settings.json`
3. 启动应用：`python -m frontend.app`
4. 切换到 MCP Tab，点击 Connect 连接服务器
5. 工具自动发现后可启用/禁用
